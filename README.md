# mkdocs-dev
mkdocsを試す

## 公式サイト
https://squidfunk.github.io/mkdocs-material/

## ローカル実行
devcontainerを使ってローカルで実行する

## 記事作成
```bash
mkdocs new .
```

## ローカル確認
```bash
mkdocs serve
```

[http://127.0.0.1:8000/](http://127.0.0.1:8000/)にアクセス

portを指定したい場合は以下を参考
```bash
mkdocs serve --dev-addr=127.0.0.1:8080
```
